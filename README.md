# University Dashboard

Dashboard interactivo para la visualización y análisis de datos estudiantiles universitarios.

---

## Descripción

**University Dashboard** es una aplicación desarrollada en Python que permite visualizar y analizar información sobre estudiantes universitarios mediante gráficos y tablas interactivas.  
El proyecto busca facilitar la comprensión de métricas como:
- Distribución de estudiantes por facultad o programa.
- Rendimiento académico general.
- Evolución de inscripciones.
- Comparaciones estadísticas y visuales entre diferentes grupos.

El propósito principal es brindar una herramienta visual para administradores, docentes o analistas de datos institucionales que deseen comprender el comportamiento académico de la comunidad estudiantil.

---

## Características principales

- Lectura y procesamiento de datos desde `university_student_data.csv`.  
- Análisis y visualización con librerías como **pandas**, **matplotlib**, **seaborn** o **plotly**.  
- Interfaz web implementada mediante el archivo `app.py` (Flask o Streamlit).  
- Ejecución sencilla en entorno local.  
- Notebook de referencia (`Data_Visualization_and_Dashboard_Deployment.ipynb`) para exploración manual y desarrollo.

---
